
## 在线文档格式转换
项目基于PHP Laravel 8框架开发，通过 Libreoffice 命令转换文档格式

支持格式：
- WORD
- PPT
- PDF
- EXCEL

word转pdf、word转ppt、ppt转pdf、ppt转word、pdf转word、pdf转ppt等

后续准备支持图片转word、图片去背景转文档功能。
